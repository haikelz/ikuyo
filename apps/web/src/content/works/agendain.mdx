## Deskripsi

Agendain adalah sebuah Website sederhana untuk membuat dan memanage agenda. Untuk datanya sendiri, masih disimpan ke Local Storage.

Di project ini, saya mengimplementasikan beberapa hal yang belum pernah saya pakai seperti [Flowbite](https://flowbite.com/) sebagai UI Components, [Clerk](https://clerk.com/) untuk me-manage User Authentication, serta mengimpelentasikan [cva(class-variance-authority)](https://cva.style/docs).

## Fitur

- User Authentication(Sign In and Sign Out with Github or Email).
- Create agenda, edit agenda, delete agenda, archive agenda, Search agenda and archive, and mark agenda as done.
- Simple page transitions and animations.
