## Description

Ryuko Matoi is a custom-built WhatsApp bot created for personal and group use.

## Key Features

- Create Sticker from image file.
- Ask to AI.
- Random Anime Quotes.
- Random jokes (more spesifically father jokes).
- Prayer schedule.
- Generate a random daily do'a.
- Edit background color of a photo.
- Optical Character Recognition (OCR) from input image.
- Asma'ul Husna

## Available Commands

- !ask
- !salam
- !info
- !editbackground
- !jadwalsholat
- !jokes
- !animequote
- !doa
- !ocr

## Technologies

- whatsapp-web.js
- Typescript
- Integration with Gemini, and Remove BG.
- Pino Logger for better debugging
