## Description

The Sugiharto adalah sebuah design yang dibuat oleh bang [Nauval Azhar](https://www.youtube.com/@mhdnauvalazhar). Disini saya menslicing design tersebut menjadi sebuah Website yang dapat diakses siapa saja.

Pada slicing kali ini saya memanfaatkan Framer Motion untuk animasi. dan membuat beberapa page sesuai dengan design yang ada.

Selain itu, saya juga mencoba menerapkan prinsip Atomic Design, yakni dimana komponen dibagi-bagi sekecil mungkin berdasarkan hierarkinya. Ada `atoms`, `molecules`, `organisms`, dan `templates`/`layout`. Bagi yang di SMA/SMK nya belajar biologi, pasti kenal dengan beberapa istilah tadi.

## Tech Stack

- Next JS
- Typescript
- Tailwind CSS
- Framer Motion
