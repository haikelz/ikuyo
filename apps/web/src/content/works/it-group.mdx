## Deskripsi

IT Group adalah salah satu grup IT yang ada di Whatsapp, yang dibuat oleh mas Vian. Disini, saya menslicing hasil UI Designnya mas Vian menjadi sebuah Website yang dapat diakses siapa saja. Di project ini saya belajar mengkonfigurasi absolute import di Vite, juga membuat reusable component.

**Note:** saya udah izin ke mas Vian untuk membagikan hasil slicingnya ke Github.
